FAQ

Preguntas frecuentes

Todo lo que necesitas saber sobre Octopus.

Octopus es una plataforma empresarial de inteligencia artificial que conecta conversaciones, sistemas y procesos de negocio. Permite unificar todos los canales de comunicación — WhatsApp, email, web, voz — y conectarlos con ERP, CRM y sistemas internos para automatizar operaciones completas.
Sí. Octopus es Tech Provider oficial de Meta para WhatsApp Business API — lo que significa que operamos directamente sobre la infraestructura de Meta para entregar mensajes de forma confiable y a escala. Esto incluye soporte para mensajes de plantilla, botones interactivos, listas, documentos, imágenes y todas las funcionalidades avanzadas de la plataforma. Puedes conocer más sobre qué significa ser Tech Provider en la documentación oficial de Meta: developers.facebook.com — Tech Providers →
Absolutamente. Octopus tiene conectores nativos para SAP Business One, SAP S/4HANA, Oracle ERP, Microsoft Dynamics y otros sistemas empresariales. La integración es bidireccional — Octopus puede consultar y escribir datos en el ERP en tiempo real durante las conversaciones.
Sí. Octopus expone una API REST completa con autenticación OAuth2, webhooks configurables y documentación detallada. Permite integrar Octopus con cualquier sistema o herramienta de tu stack tecnológico.
Ese es el modelo de Octopus: no reemplaza tus sistemas existentes, los conecta y potencia. SAP sigue siendo tu ERP, Salesforce sigue siendo tu CRM — Octopus actúa como el orquestador inteligente que los une con tus conversaciones.
Sí. Octopus soporta arquitecturas multi-tenant con completo aislamiento de datos entre empresas o marcas. Ideal para grupos corporativos, franquicias, o empresas con múltiples líneas de negocio.
El Automation Engine de Octopus permite definir flujos de trabajo visuales que se activan por eventos de conversación. Por ejemplo: cuando un cliente solicita una cotización por WhatsApp, el flujo consulta el ERP, genera la cotización, la envía al cliente y actualiza el CRM — todo sin intervención humana.
